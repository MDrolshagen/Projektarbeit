Hobby

Mein Hobby ist das Programmieren von Anwendungen für die unterschiedlichtsten Betriebssysteme. Dafür benutze ich Unity. Eine Game-Engine, die eine breite Auswahl an Funktionen und Features bietet.
Als Programmiersprache in Unity wird C# genutzt.

C# ist ein passende Sprache dafür und vor allem auch einfach zu lernen.
Unity bietet außerdem einen Marketplace oder Asset Store. Dieser ist eine Plattform auf der vorgefertigte Assets von anderen Entwicklern kostenlos oder gegen Geld angeboten werden.
Durch den Asset-Store wird das Entwickeln vereinfacht und bietet auch eine gewisse Inspiration.

Es gibt auch andere bekannte Game-Engines wie z. B. „Unreal Engine”, „Godot” oder „CryEngine”
